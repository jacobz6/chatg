# 《高校研究生秘书》游戏架构文档

## 一、游戏概述

### 1.1 基本信息
- **游戏名称**：高校研究生秘书
- **游戏类型**：视觉小说 / 文字冒险 / 剧情向
- **游戏平台**：Ren'Py
- **输出格式**：剧本大纲 → 章节剧本(.txt) → Ren'Py代码(.rpy)

### 1.2 核心主旨
讲述一位31岁高校研究生秘书在职场中经历的三次重大抉择。三章内容相互独立又暗藏伏笔，玩家的选择将影响主角的性格数值与价值判断，最终导向截然不同的十种结局。旨在探讨：正义与生存的博弈、信任的建立与崩塌、以及职场PUA环境下的自我保护。

### 1.3 目标受众
- 对职场剧情、现实主义题材感兴趣的玩家
- 喜欢多分支、多结局的剧情探索型玩家
- 对现实教育体制、职场生态有反思的玩家

---

## 二、世界观设定

### 2.1 背景介绍
故事发生在一所综合性大学的研究生院。主角「沈思」是研究生院的行政秘书，31岁，工作三年，负责处理学生事务、报销审核、档案管理等日常繁琐工作。

### 2.2 核心场景
| 场景 | 说明 |
|------|------|
| 研究生院办公室 | 主角日常工作的场所，开放式工位 |
| 学院会议室 | 报销审批、学术委员会会议 |
| 教师休息室 | 非正式交谈、消息集散地 |
| 学生事务中心 | 学生提交材料、咨询的窗口 |
| 学院资料室 | 档案存放、调阅审批 |

### 2.3 时间线
- **学期中段**（第三章故事发生时间点往前推2个月）：新学期开学
- **学期末**（第一章）：毕业论文提交截止期前后
- **暑假前**（第二章）：财务报销密集期
- **新学年开始前**（第三章）：评优晋升阶段

## 四、章节大纲

### 4.1 第一章：论文风云

**时间**：学期末 · 毕业论文提交截止期前三天

**主线剧情**：
林子轩因找工作耽误了论文进度，提交超期。他伪造急性阑尾炎病历，希望主角「网开一面」帮他赶上答辩末班车。主角明察秋毫，识破伪造，通过正规流程拒绝了请求。

**剧情走向**：
1. 林子轩提交超期申请 → 主角按规定拒绝
2. 林子轩出示「病历证明」→ 主角发现破绽
3. （分支A）坚持原则上报 → 事件平息
4. （分支B）私下提醒「走正规流程申诉」→ 埋下隐患
5. 一个月后：林子轩因无法毕业失去高薪offer
6. （分支）林子轩自杀未遂/成功/被救 → 主角心理创伤

**关键抉择点**：
| 抉择 | 选项 | 影响 |
|------|------|------|
| 面对林子轩的请求 | A. 直接拒绝 B. 表示同情但仍拒绝 C. 提出替代方案 | +责任值/+洞察值 |
| 发现病历伪造时 | A. 当场揭穿 B. 私下询问 C. 假装不知道 | +洞察值/+信任值 |
| 事后面对学生议论 | A. 保持沉默 B. 解释情况 C. 自我怀疑 | 影响后续心理状态 |

**数值影响**：
- `responsibility +1`：坚持原则
- `insight +1`：识破伪造
- `trust -1`：被部分学生误解
- `chapter1_choice_record`：记录关键选择

---

### 4.2 第二章：账目迷局

**时间**：暑假前 · 财务报销密集期

**主线剧情**：
主角在日常报销审核中发现陈明德教授一笔异常报销，数额巨大且票据存疑。主角面临艰难抉择：提醒、沉默、还是举报。

**剧情走向**：
1. 发现异常票据 → 金额、时间、供应商均存疑
2. （分支A）直接举报 → 领导压下、主角被打压
3. （分支B）暗示教授 → 教授反邀共犯、主角陷入泥潭
4. （分支C）装傻不知 → 良心不安、另寻出路
5. 教授发现主角知情 → 威逼利诱
6. （接受/拒绝）→ 后果截然不同
7. 拒绝后被边缘化：工位调换、权限削减、孤立

**关键抉择点**：
| 抉择 | 选项 | 影响 |
|------|------|------|
| 发现异常时 | A. 立刻举报 B. 暗中调查 C. 假装没看见 | 开启不同路线 |
| 教授来电时 | A. 直接拒绝 B. 委婉推脱 C. 假意答应 | 影响后续对抗策略 |
| 领导施压时 | A. 坚持立场 B. 妥协退让 C. 寻找盟友 | 结局走向 |

**数值影响**：
- `insight +2`：发现真相
- `crossed_boundary`：是否越界
- `chose_institution`：是否选择体制

---

### 4.3 第三章：姐妹迷局

**时间**：新学年开始 · 年度评优阶段

**主线剧情**：
新来的同事周雅琴对主角百般照顾，主角逐渐打开心扉。然而主角发现周雅琴与院长存在不正当关系，并在评优时发现周雅琴窃取了她的工作成果。

**剧情走向**：
1. 周雅琴入职 → 热心帮助、逐渐熟络
2. 主角发现办公室恋情 → 震惊、犹豫是否告知
3. （分支A）直接质问 → 关系破裂、被反咬
4. （分支B）暗中收集证据 → 持久战、消耗心力
5. （分支C）隐忍不发 → 周雅琴愈发嚣张
6. 年度评优：周雅琴窃取主角材料 → 主角落选
7. （分支）主角反击：散播消息 → 证据不足、被围攻
8. 结局：主角精神出问题 / 绝地反击 / 自我和解

**关键抉择点**：
| 抉择 | 选项 | 影响 |
|------|------|------|
| 发现恋情时 | A. 当面质问 B. 假装没看见 C. 暗示周雅琴 | 不同关系走向 |
| 评优材料被窃时 | A. 忍气吞声 B. 找领导申诉 C. 散播消息 | 不同结局 |
| 被众人围攻时 | A. 坚持到底 B. 妥协道歉 C. 离开体制 | 最终命运 |

**数值影响**：
- `trust +2`：与周雅琴建立信任
- `betrayed`：是否被背叛
- `evidence_collected`：是否收集证据

## 五、分支系统设计

### 5.1 核心数值体系

| 数值 | 说明 | 影响 |
|------|------|------|
| `integrity` 正直值 | 坚持原则的程度 | 导向正义结局 |
| `insight` 洞察值 | 明辨是非的能力 | 影响判断准确性 |
| `resilience` 韧性值 | 抗压能力 | 影响是否崩溃 |
| `social` 人际关系 | 与同事/领导的亲疏 | 影响资源获取 |

### 5.2 章节独立分支

每章独立的分支变量（不影响其他章节，但影响最终汇总）：

```
# 第一章
ch1_integrity_choice    # 是否坚持原则
ch1_empathy_choice      # 是否展现同理心
ch1_aftermath_handling  # 如何处理后果

# 第二章
ch2_report_choice       # 是否举报
ch2_resist_temptation   # 是否抵抗诱惑
ch2_support_accepted   # 是否接受帮助

# 第三章
ch3_trust_to_zhou       # 对周雅琴的信任程度
ch3_evidence_gathered  # 是否收集证据
ch3_retaliation_style  # 反击方式
```

## 六、十种结局设计

### 6.1 结局分类

| 类型 | 数量 | 说明 |
|------|------|------|
| 正面结局 | 2 | 坚守本心、有所成长 |
| 负面结局 | 4 | 职场失败或精神崩溃 |
| 中立结局 | 2 | 离开体制、重新开始 |
| 隐藏结局 | 2 | 特殊选择触发 |

### 6.2 结局详解

#### 【结局一】理想结局·破茧
**触发条件**：
- 三章均选择正直选项
- `integrity >= 7` 且 `resilience >= 5`
- 收集到关键证据但未滥用

**结局概述**：
主角在经历三次事件后，虽然历经挫折，但始终保持初心。最终其坚持得到认可，被调任至学校纪检部门，成为维护公平的存在。林子轩自杀未遂后在心理干预下逐渐康复，陈明德终因证据充分被查处，周雅琴事件也得到公正处理。

#### 【结局二】理想结局·重生
**触发条件**：
- 第二章举报被压下后不离不弃
- 第三章发现证据但走正规途径
- `insight >= 6` 且 `social >= 5`

**结局概述**：
主角选择离开学校，跳槽至企业做合规管理。在新的环境里，她将校园经历转化为职业财富，成为受人尊敬的专业人士。回首过去，她感谢那些挫折让自己成长。

#### 【结局三】悲剧结局·沉沦
**触发条件**：
- 第一章后因学生自杀事件受到心理创伤
- 后续选择愈发保守逃避

**结局概述**：
主角从此畏畏缩缩，不敢再做任何决定，工作频繁出错，最终被调离核心岗位，在档案室度过余生。

#### 【结局四】悲剧结局·黑化
**触发条件**：
- 第二章接受教授诱惑尝到甜头
- 第三章以谣言反击失败

**结局概述**：
主角逐渐变成自己曾经讨厌的人，善于利用规则漏洞，在体制内如鱼得水，但夜深人静时常常失眠。

#### 【结局五】悲剧结局·崩溃
**触发条件**：
- 第三章遭遇背叛和围攻
- `resilience < 4`

**结局概述**：
主角确诊抑郁症，不得不离职治疗。在心理医生的帮助下慢慢恢复，但再也无法回到正常工作状态。

#### 【结局六】悲剧结局·玉石俱焚
**触发条件**：
- 第三章强行散播谣言被反噬
- 愤而做出过激行为

**结局概述**：
主角因散布不实信息被起诉，身败名裂。最终虽证明了周雅琴的问题，但也付出了沉重代价。

#### 【结局七】中立结局·离去
**触发条件**：
- 三个事件中任意两个选择逃避或妥协

**结局概述**：
主角身心俱疲，选择辞职考研离开教育行业。新的生活未必更好，但至少不用再面对那些人。

#### 【结局八】中立结局·独善
**触发条件**：
- 始终保持中立态度
- 不主动得罪人也不主动帮忙

**结局概述**：
主角成为一个精致的利己主义者，安全地在体制内生存下去，但也失去了真正的朋友和自我。

#### 【结局九】隐藏结局·觉醒（正面）
**触发条件**：
- 第一章展现高度同理心（不是懦弱）
- 第二章拒绝诱惑但巧妙保护自己
- 第三章不被周雅琴迷惑但也不走向极端
- `insight >= 8`

**结局概述**：
主角意识到真正的敌人不是某个人，而是整个病态的体制。她选择站出来，用法律和媒体而非体制内部规则来解决问题。

#### 【结局十】隐藏结局·轮回（负面）
**触发条件**：
- 第一章学生自杀
- 第二章举报被压
- 第三章被同事背叛
- 但每次都选择隐忍而非爆发

**结局概述**：
多年后，主角成为研究生院主任，面对同样的事件，她做出了和当年领导一样的选择——压下去。游戏到此结束，暗示这种悲剧不会终结。

## 七、剧本格式规范

### 7.1 章节剧本模板

```
# 《高校研究生秘书》第一章：论文风云

## 场景1：办公室·上午

【场景描述】
研究生院办公室，清晨的阳光透过百叶窗在地面投下条纹。键盘敲击声和打印机嗡鸣声交织，沈思正在处理毕业材料审核。

【出场人物】
- 沈思（主角）：（动作）低头翻阅文件
- 林子轩（学生）：（动作）敲门进入，神色焦急

---

林子轩：沈老师，我...我有件事想跟您商量。

> 选择分支
> A. "请坐，什么事？"（温和回应）
> B. "我正在忙，等一下。"（冷淡回应）
> C. （放下手中的工作看向他）（关注）

---

【分支A：温和回应】

林子轩：是这样的，我的论文...提交超期了，但我真的有特殊情况...

沈思：（内心独白）超期？这可是大事...

> 选择分支
> A. "按规定，超期申请需要院长审批，而且..."（直接拒绝）
```
