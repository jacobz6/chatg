# 🔧 快速测试和解决方案

## 📢 问题：人物依旧很大

### 方案1：使用极端小的缩放测试（推荐）

**在剧本中直接测试：**
```renpy
# 首先尝试这个 - 非常小的缩放0.2，一定能看到完整人物
$ show_char("shen_si", "normal", position="left", zoom=0.2)

# 如果0.2可以，再试试0.3
$ show_char("shen_si", "normal", position="left", zoom=0.3)

# 逐渐增加到合适的大小
$ show_char("shen_si", "normal", position="left", zoom=0.4)
$ show_char("shen_si", "normal", position="left", zoom=0.5)
```

---

### 方案2：直接使用Ren'Py原生语法（最可靠）

**如果上面的函数还是不行，直接用这个：**
```renpy
# 最直接的方式 - 完全绕过辅助函数
show shen_si_normal:
    xpos 0.15
    ypos 1.0
    xanchor 0.5
    yanchor 1.0
    zoom 0.4  # 这里直接改缩放值
```

**其他位置：**
```renpy
# 中间
show shen_si_normal:
    xpos 0.5
    ypos 1.0
    xanchor 0.5
    yanchor 1.0
    zoom 0.4

# 右边
show shen_si_normal:
    xpos 0.85
    ypos 1.0
    xanchor 0.5
    yanchor 1.0
    zoom 0.4
```

---

### 方案3：调整全局默认缩放

**修改 `characters.rpy` 第136行：**
```renpy
# 原来是0.4
define char_zoom = 0.3  # 改小一点
# 或者
define char_zoom = 0.2  # 更小
```

---

## 📊 推荐的缩放测试顺序

| 测试顺序 | 缩放值 | 预期效果 |
|----------|--------|----------|
| 第1次 | 0.2 | 人物很小，一定完整 |
| 第2次 | 0.3 | 人物偏小 |
| 第3次 | 0.4 | 人物适中 |
| 第4次 | 0.5 | 人物稍大 |
| 第5次 | 0.6 | 人物较大 |

**找到合适的大小后，设置为全局默认：**
```renpy
define char_zoom = 0.4  # 这里改成你找到的合适值
```

**在剧本中添加调试信息：**

# 先显示一个简单的背景看是否正常

$ show_char("shen_si", "normal", position="left", zoom=0.2)

# 慢慢调大

## 💡 关于Ren'Py更新

**不需要更新Ren'Py！** 这个问题与Ren'Py版本无关，是：
1. 图片尺寸（1440px）> 屏幕高度（1080px）
2. 需要正确的缩放配置

---

## 🚨 如果以上都不行

**检查以下几点：**

1. **图片文件名是否正确？**
   - 应该是：`images/chars/shen_si_normal.png`

2. **图片尺寸是1080×1440吗？**
   - 宽度：1080px
   - 高度：1440px

3. **用系统的图片查看器打开看看图片是否正常？**

## 📝 快速测试剧本示例

**创建一个简单的测试脚本 `test_char.rpy`：**
```renpy
label test_chars:
    scene black
    
    "测试开始！"
    
    "=== 测试1：非常小的缩放0.2 ==="
    $ show_char("shen_si", "normal", position="left", zoom=0.2)
    "能看到完整的小人吗？"
    $ hide_char("shen_si")
    
    "=== 测试2：缩放0.3 ==="
    $ show_char("shen_si", "normal", position="left", zoom=0.3)
    "怎么样？"
    $ hide_char("shen_si")
    
    "=== 测试3：缩放0.4 ==="
    $ show_char("shen_si", "normal", position="left", zoom=0.4)
    "这个大小合适吗？"
    $ hide_char("shen_si")
    
    "测试结束！"
    return
```

然后在游戏中调用 `jump test_chars` 来测试！
