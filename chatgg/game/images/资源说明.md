# 图片资源说明

## 文件夹结构

```
images/
├── bg/      # 场景背景
├── chars/   # 角色立绘
└── props/   # 道具/UI
```

## 图片命名规范

### 背景图片 (images/bg/)
- 格式：`bg_场景名.png`
- 示例：
  - `bg_office.png`
  - `bg_office_morning.png`
  - `bg_canteen.png`
- 尺寸：1920×1080 (16:9)

### 角色立绘 (images/chars/)
- 格式：`角色名_表情.png`
- 示例：
  - `shen_si_normal.png`
  - `lin_zixuan_nervous.png`
- 尺寸：1080×1440 (高度占满画面，透明背景)

### 道具图片 (images/props/)
- 格式：`props_道具名.png`
- 示例：
  - `props_paper_diagnosis.png`
  - `props_paper_crumpled.png`
- 尺寸：800×600 (适中)

## 在剧本中使用图片

### 1. 显示背景
```renpy
# 方法1：使用辅助函数（推荐）
$ show_bg("office_morning")

# 方法2：直接使用Ren'Py语句
scene bg_office_morning with fade
```

### 2. 显示角色立绘
```renpy
# 方法1：使用辅助函数（推荐）
$ show_char("shen_si", "normal", position="left")

# 方法2：直接使用Ren'Py语句
show shen_si_normal at left with dissolve
```

### 3. 显示道具
```renpy
# 方法1：使用辅助函数（推荐）
$ show_prop("paper_diagnosis", position="right")

# 方法2：直接使用Ren'Py语句
show props_paper_diagnosis at right with dissolve
```

### 4. 隐藏图片
```renpy
# 隐藏角色
$ hide_char("shen_si")

# 隐藏道具
hide props_paper_diagnosis
```

## 临时占位方案

在正式图片完成前，可以使用纯色图片或临时图片作为占位：
1. 使用任意图片重命名为需要的文件名
2. 放入对应的文件夹
3. Ren'Py会自动加载

## 图片格式要求

- **推荐格式**：PNG (支持透明背景)
- **备用格式**：WebP (文件更小)
- **颜色模式**：RGB或RGBA
- **DPI**：72-96 (屏幕显示)

## 优先级建议

1. **最高优先级**：主角立绘、主要场景
2. **中等优先级**：其他角色立绘
3. **低优先级**：道具、次要场景

## 📐 四、图片格式规范

### 推荐格式
- **主格式**：PNG（支持透明背景）
- **备用格式**：WebP（文件更小，质量好）

### 尺寸规范
| 类型 | 推荐尺寸 | 说明 |
|------|----------|------|
| **场景背景** | 1920×1080 | 16:9 全高清 |
| **角色立绘** | 1080×1440 | 人物高度占满画面，保留透明背景 |
| **UI元素** | 按需设计 | 按现有GUI框架尺寸 |
| **道具图片** | 800×600 | 适中大小 |

### 图片命名规范
```
# 角色立绘
角色名_表情.png
例如：shen_si_normal.png, lin_zixuan_nervous.png

# 场景背景
bg_场景名_状态.png
例如：bg_office_morning.png, bg_canteen.png

# 道具/UI
用途_描述.png
例如：paper_diagnosis.png, gui_textbox.png
```

### 艺术风格建议
- **风格**：二次元卡通风格，偏写实（不要太Q版）
- **色调**：清新自然，符合高校氛围
- **光线**：注意不同场景的光影变化（清晨、下午、夜晚）
