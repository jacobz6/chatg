# Daily Dialogue

日常对话设计技能，专注于生成自然、逻辑连贯、符合角色设定的日常对话。

## 核心原则

### 1. 逻辑连贯性

- 对话内容必须与上下文紧密衔接
- 避免突然的话题跳转
- 回应要有针对性，不能答非所问
- 保持时间线逻辑一致

### 2. 角色一致性

- 说话风格符合角色设定
- 用词习惯与角色背景匹配
- 情感表达符合角色性格
- 称呼和关系用语准确

### 3. 对话自然度

- 符合真实对话的节奏
- 适当的省略和口语化表达
- 正常的互动回合
- 自然的情感起伏

## 对话类型

### 日常寒暄

```
A：早啊，今天气色不错。
B：嗯，睡了个好觉。你怎么也这么早？
A：赶作业呗，deadline是第一生产力。
B：这话听着怎么这么耳熟...
```

### 日常请求

```
A：打扰一下，能帮个忙吗？
B：什么事？
A：帮我看看这段代码哪儿错了？
B：让我看看...哦，这儿少了个分号。
```

### 日常争论

```
A：你怎么能这么说！
B：我只是在陈述事实。
A：事实也不是你这样说的啊！
B：好好好，我错了，行了吧。
```

### 日常关心

```
A：最近看你好像很累的样子。
B：嗯，项目快上线了，天天加班。
A：别太拼了，身体要紧。
B：知道了，谢啦。
```

## 工作流程

### 1. 明确场景要素

| 要素 | 说明 |
|------|------|
| 场景 | 在哪里发生的对话 |
| 人物 | 谁在对话（2人或多人） |
| 关系 | 对话者之间的关系 |
| 目的 | 这次对话要达成什么 |
| 情绪 | 当前整体情绪氛围 |

### 2. 构建对话框架

```
开场 → 引入话题 → 信息交换 → 情感互动 → 收尾
```

### 3. 检查逻辑链条

```
前一句 ←→ 后一句
   ↓
是否衔接？
   ↓
是否符合角色设定？
   ↓
是否符合场景逻辑？
```

## 角色对话模板

### 学生角色

```
[性格：开朗]
称呼：昵称/同学
语气：轻松、活泼
常用词：绝了、yyds、666、笑死

[性格：内向]
称呼：姓名/学长学姐
语气：轻柔、犹豫
常用词：嗯...那个...可能吧
```

### 职场角色

```
[性格：严肃]
称呼：职称/姓名
语气：正式、简洁
常用词：关于...我认为...需要确认

[性格：亲和]
称呼：名字/昵称
语气：轻松、友好
常用词：好的呀、没问题、包在我身上
```

### 家庭角色

```
[长辈]
称呼：爸妈/爷爷奶奶
语气：关怀、叮嘱
常用词：吃饭了吗/早点休息/别太累

[晚辈]
称呼：直接叫称谓
语气：撒娇/调皮/尊敬
常用词：好不好嘛/求你了/知道啦
```

## 场景示例

### 场景1：宿舍早晨

**人物**：室友A（起床困难户）、室友B（早起达人）
**情境**：B准备出门，A刚醒

```
B：我要去上课了，你今天不是也有课吗？
A：（揉眼睛）几点了...
B：都八点了，第一节九点。
A：啊啊啊怎么不早叫我！（弹起来）
B：叫了三次了你都不醒，还说我吵你睡觉。
A：行行行我错了，让我穿个衣服...
B：快点啊，帮你在食堂带个包子？
A：牛肉的！谢啦！
B：得嘞，先走了！
```

### 场景2：餐厅点餐

**人物**：顾客、服务员
**情境**：顾客纠结菜单

```
服务员：您好，请问几位？
顾客：两位。
服务员：好的，请问需要点些什么？
顾客：有什么推荐的吗？
服务员：招牌的宫保鸡丁和糖醋排骨都很受欢迎。
顾客：那就一个宫保鸡丁，一个...红烧肉吧。
服务员：好的，还需要别的吗？
顾客：再来两碗米饭，一杯柠檬水。
服务员：好的，请稍等。
```

### 场景3：朋友约会

**人物**：闺蜜A、闺蜜B
**情境**：讨论周末计划

```
A：周末有空吗？
B：周六可以，周日要加班。
A：那周六出来逛街呗！听说新开了家网红店。
B：行啊，几点？
A：下午两点，老地方见？
B：没问题，正好我也想去看看那家店。
A：那我先做做功课，看看还有什么好逛的。
B：好嘞，到时候发我。
A：没问题！
```

## 输出格式

### 标准格式

```markdown
## 场景：XXX

### 人物
- A（角色名）：[性格特点]
- B（角色名）：[性格特点]

### 对话

A：[开场白]
B：[回应]
A：[追问/反应]
B：[回应]
...
```

### 精简格式

```
A：XXX
B：XXX
A：XXX
B：XXX
```

## 注意事项

- 避免对话过于完美，要有正常的停顿、犹豫、转移话题
- 适当加入动作描写和语气词增强画面感
- 注意对话的节奏感，有来有往
- 保持人物语言风格的一致性
- 对话要推动情节或展现人物关系
- 避免大段独白，日常对话以短句为主
